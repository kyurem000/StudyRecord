비트맵(Bitmap) 비트맵은 픽셀이 모여 만들어진 정보의 집합으로 레스터(Raster)이미지라고도 부른다. 픽셀 단위로 화면에 렌더링한다. ( 렌더링 : 컴퓨터가 화면에 그림을 그려서 우리가 볼 수 있게 합니다 ) 벡터(Vector)는 수학적 정보의 형태(Shape)들이 만들어내는 결과물입니다. 이미지의 점, 선, 면의 위치(좌표), 색상 등의 정보를 온전히 가지고 있으며 그를 화면에 렌더링합니다. 수학적 정보만을 가지고 있기 때문에 이미지 확대/축소에 따른 용량 변화가 없다. 비트맵 장점 : 정교하고 다양한 색상을 자연스럽게 표현 비트맵 단점 : 확대/축소시 계단 현상, 품질 저하 벡터 장점 : 확대/축소에서 자유로움, 용량변화가 없음 벡터 단점 : 정교한 이미지(인물, 풍경 사진 등) 를 표현하기 어려움 JPG(JPEG) 손실 압축 표현 색상도가 뛰어나 고해상도 표시장치에 적합함 (24비트컬러, 약 1600만 색상 지원) 이미지 품질과 용량이 쉽게 조절 가능 가장 널리 쓰이는 이미지 포맷 높은 압축률(적은 용량) PNG : gif의 대체 포맷으로 개발됨 비손실 압축 8비트
